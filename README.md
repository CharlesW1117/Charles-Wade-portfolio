## 🚀 Featured Projects

### 🎯 Whack-A-Mole Game
An interactive browser game built with React featuring randomized mole movement, score tracking, and restart logic.  
Solved race-condition bugs by restructuring the game loop using controlled intervals and centralized Context API state.  
**Stack:** React · Context API · State Management  
🔗 [GitHub](https://github.com/CharlesW1117/Whack-a-mole) · [Live Demo](https://charlesw1117.github.io/Whack-a-mole/)

A collaborative full-stack fitness application with JWT authentication, REST API integration, and PostgreSQL data persistence.
